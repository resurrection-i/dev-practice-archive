## 📖 系统简介

校园公告信息管理系统是一个基于Spring Boot + MyBatis + Thymeleaf开发的Web应用，采用粉色梦幻主题设计，提供公告发布、分类管理、标签管理、评论互动、用户管理等功能。

**主要特性：**
- 🌸 粉色梦幻主题，温馨浪漫
- 📢 公告发布与管理
- 💬 评论互动功能
- ⭐ 收藏功能
- 👥 用户权限管理
- 📊 数据统计分析

## 🔧 快速安装

### 1. 验证环境

打开命令行（CMD或PowerShell），执行以下命令验证环境：

```bash
# 检查Java版本
java -version
# 应该显示：java version "1.8.0_xxx" 或更高

# 检查Maven版本
mvn -v
# 应该显示：Apache Maven 3.6.x 或更高

# 检查MySQL版本
mysql --version
# 应该显示：mysql Ver 8.0.x
```

### 2. 下载项目

```bash
# 如果使用Git
git clone <项目地址>

# 或者直接下载ZIP包并解压
```

### 3. 进入项目目录

```bash
cd campus-notice-system
```

## ⚙️ 项目配置

### 修改数据库连接

编辑 `src/main/resources/application.yml` 文件：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/campus_notice?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai
    username: root          # 修改为你的MySQL用户名
    password: your_password # 修改为你的MySQL密码
    driver-class-name: com.mysql.cj.jdbc.Driver
```

**重要：** 请将 `your_password` 替换为你的MySQL密码！

### 文件上传路径配置（可选）

如果需要修改文件上传路径，编辑 `application.yml`：

```yaml
file:
  upload:
    path: C:/uploads/  # Windows系统
    # path: /var/uploads/  # Linux/Mac系统
```

### 方式一：使用Maven命令（推荐）

```bash
# 在项目根目录执行
mvn clean install
mvn spring-boot:run
```

1. 用IntelliJ IDEA打开项目
2. 等待Maven依赖下载完成
3. 找到 `src/main/java/com/campus/notice/CampusNoticeApplication.java`
4. 右键 → Run 'CampusNoticeApplication'

## ❓ 常见问题

### 1. 启动失败：端口被占用

**错误信息：**
```
Port 8080 was already in use
```

**解决方案：**

**方式A：** 修改端口（推荐）

编辑 `application.yml`：
```yaml
server:
  port: 8081  # 改为其他端口
```

**方式B：** 关闭占用端口的程序

```bash
# Windows
netstat -ano | findstr :8080
taskkill /PID <进程ID> /F

# Linux/Mac
lsof -i :8080
kill -9 <进程ID>
```

### 2. 数据库连接失败

**错误信息：**
```
Communications link failure
```

**解决方案：**

1. **检查MySQL是否启动**
   ```bash
   # Windows
   net start mysql
   
   # Linux/Mac
   sudo systemctl start mysql
   ```

2. **检查用户名密码**
   - 确认 `application.yml` 中的用户名和密码正确

3. **检查数据库是否存在**
   ```sql
   SHOW DATABASES LIKE 'campus_notice';
   ```

### 3. Maven依赖下载失败

**解决方案：**

1. **配置国内镜像**

编辑 `~/.m2/settings.xml`（如果不存在则创建）：

```xml
<settings>
  <mirrors>
    <mirror>
      <id>aliyun</id>
      <mirrorOf>central</mirrorOf>
      <name>Aliyun Maven</name>
      <url>https://maven.aliyun.com/repository/public</url>
    </mirror>
  </mirrors>
</settings>
```

2. **清理并重新下载**
   ```bash
   mvn clean
   mvn install
   ```

### 4. 文件上传失败

**错误信息：**
```
FileNotFoundException
```

**解决方案：**

1. **创建上传目录**
   ```bash
   # Windows
   mkdir C:\uploads
   
   # Linux/Mac
   sudo mkdir -p /var/uploads
   sudo chmod 777 /var/uploads
   ```

2. **检查配置**
   - 确认 `application.yml` 中的 `file.upload.path` 路径存在

### 5. 页面样式错误

**解决方案：**

1. **清除浏览器缓存**
   - 按 `Ctrl + F5` 强制刷新

2. **检查静态资源**
   - 确认 `src/main/resources/static` 目录存在

### 6. 中文乱码

**解决方案：**

1. **设置IDE编码**
   - IntelliJ IDEA: Settings → Editor → File Encodings → UTF-8

2. **设置MySQL编码**
   ```sql
   ALTER DATABASE campus_notice CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
   ```
