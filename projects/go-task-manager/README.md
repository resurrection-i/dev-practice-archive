# Go 任务管理系统

基于 **Go + Gin + GORM + MySQL** 开发的任务管理练习项目，支持用户鉴权、RBAC 权限控制、任务创建分配与状态流转。

---

## 项目简介

这个项目主要用于练习 Go Web 后端开发的基础能力，围绕“用户管理 + 任务管理 + 权限控制”构建了一套较完整的后台服务。

项目包含：

- 用户注册与登录
- JWT 鉴权
- 角色权限控制
- 任务创建、分配、更新与删除
- 任务历史记录
- 简单前端页面

整体更偏后端实践型项目，适合用来展示 Go 在业务系统开发中的基本能力。

## 快速开始

### 环境要求

- Go 1.21+
- MySQL 5.7+

### 配置项目

将配置文件按本地环境调整：

- 使用 `config.example.yaml` 作为参考
- 配置数据库连接信息
- 配置 JWT 密钥和日志路径

### 启动项目

```bash
go mod download
go build
./go-task-manager
```

## 接口能力

主要包含三类接口：

- 用户接口：用户信息查询与管理
